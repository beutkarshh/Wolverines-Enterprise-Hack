System Architecture
🧠
Google Gemini 2.0 Flash
Powers Priya's conversation intelligence. Handles multilingual understanding, intent detection, and knowledge-based responses about MHT-CET and Campus Dekho services.
AI Engine
🎙️
ElevenLabs TTS
Converts Priya's responses into natural speech. Three distinct voice profiles — Payal (EN/HI) and a dedicated Marathi voice for authentic regional communication.
Voice Synthesis
📡
Exotel Telephony
Enterprise telephony API connecting AI to real phone numbers. Handles outbound dialing, IVR flows, call recording, and status webhooks across India.
Real Calls
⚡
Node.js + WebSocket
Real-time bidirectional communication between the AI engine and dashboard. PM2 cluster mode ensures zero-downtime deployments and auto-recovery.
Backend
🗄️
SQLite Knowledge Base
Structured storage for FAQs, counselling packages, event schedules, and call logs. Enables Priya to access precise event details and track conversation history.
Database
📱
Twilio WhatsApp
Post-call engagement via WhatsApp — sends registration links, event details, counselling package info, and document checklists automatically after inbound calls.
Messaging
